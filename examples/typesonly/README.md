# Types Only Example

Generate TypeScript types from Go without using tygor as a framework.

## Features

| Go | TypeScript |
|----|------------|
| `struct` with nested types | Interfaces with automatic dependency resolution |
| `*T` pointer fields | `T \| undefined` optional fields |
| `const` groups (`type Role string`) | String union types (`"admin" \| "editor"`) |
| `type Page[T any] struct` | Generic interfaces (`Page<T>`) |
| `map[string]T` | `Record<string, T>` |
| `time.Time` | `string /* RFC3339 */` with type hint |
| `int`, `int32`, etc. | `number /* int */` with type hint |
| Doc comments | JSDoc comments (type and field level) |

```typescript title="types_github_com_broady_tygor_examples_typesonly_api.ts"
// Code generated by tygor. DO NOT EDIT.

/** Page is a generic paginated response. */
export interface Page<T> {
  items?: T[];
  total: number /* int */;
  hasMore: boolean;
}
/** Profile contains optional user profile information. */
export interface Profile {
  bio: string;
  avatar?: string;
  links?: Record<string, string>;
}
/** Role represents the access level of a user. */
export type Role = "admin" | "editor" | "viewer";
/** User represents a user in the system. */
export interface User {
  id: string;
  email: string;
  role: Role;
  /** Profile is nil for users who haven't completed onboarding. */
  profile?: Profile;
  tags?: string[];
  metadata?: Record<string, unknown>;
  createdAt: string /* RFC3339 */;
}
```

## Usage

<!-- [snippet:main] -->
```go title="main.go"
func main() {
	gen := flag.Bool("gen", false, "generate TypeScript types")
	out := flag.String("out", "./client/src/types", "output directory")
	flag.Parse()

	if *gen {
		generate(*out)
		return
	}

	log.Println("Run with -gen to generate TypeScript types")
}

func generate(dir string) {
	// Pass root types - referenced types are followed automatically.
	_, err := tygorgen.FromTypes(
		api.User{},
		api.Page[api.User]{}, // generic instantiation
	).ToDir(dir)

	if err != nil {
		log.Fatal(err)
	}

	log.Println("Generated types to", dir)
	os.Exit(0)
}

```
